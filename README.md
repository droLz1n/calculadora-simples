# Calculadora no Terminal com Python

Este projeto consiste em uma calculadora desenvolvida em Python, com interface no terminal utilizando a biblioteca Rich.

A aplicação permite ao usuário realizar operações matemáticas básicas de forma interativa, com um menu organizado e visualmente aprimorado, proporcionando uma melhor experiência de uso no terminal.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão (com tratamento para divisão por zero)
- Cálculo de porcentagem
- Execução contínua com opção de encerramento pelo usuário

## Interface

O projeto utiliza a biblioteca Rich para:
- Exibição de textos formatados
- Criação de menus utilizando Panel
- Melhoria na visualização de erros com traceback

## Tecnologias utilizadas

- Python 3
- Rich
